# mcat-bot

![](./assets/cat-bot-digital-logo-with-text.png)

powerful personal wechaty chatbot, which can save your time.

## Quick Start

### 1. clone

```shell script
git clone https://github.com/wj-Mcat/mcat-bot
```

### 2.requirements

```shell script
pip install -r requirements.txt
```

### 3. run bot
```shell script
python bot
```
